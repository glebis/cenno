# Code Of Conduct

cenno contributors are expected to keep project discussion direct, specific,
and respectful.

## Expected Behavior

- Focus criticism on code, design, documentation, and release risks.
- Assume maintainers and contributors are working from incomplete
  information.
- Make technical disagreement explicit and actionable.
- Respect privacy. Do not post private prompt content, answer history,
  tokens, or logs with personal data.

## Unacceptable Behavior

- Personal attacks, harassment, or discriminatory remarks.
- Publishing other people's private information.
- Sustained disruption of issues, reviews, or discussions.

## Enforcement

Violations can be reported privately to `glebis@gmail.com`. The maintainer
may edit, lock, or remove contributions and ban contributors for behavior
that violates this code.
